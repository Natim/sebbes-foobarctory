# The hard way
* [Install elm](https://guide.elm-lang.org/install/elm.html)
* Compile the project: 
   ```elm make --output=main.js  src/Main.elm```
* Launch `elm reactor`
* Browse to: `http://localhost:8000/index.html`
* Play!
